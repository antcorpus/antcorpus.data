فاز مساء اليوم  فريق تاندا الايفواري أمام آس فان من النيجر بنتيجة 3-0 ، لحساب مباراة ذهاب الدور التمهيدي الاول لدوري ابطال افريقيا وبالتالي إقترب من مواجهة النجم الساحلي في الدور 16.
في المقابل إنتهت مباراة غوريان السنغالي مع نظيره حوريا كوناكري الغيني بنتيجة 0-0، ليتأجّل الحسم بينهما الى مباراة الإيّاب في العاصمة الغينيّة كوناكري حتّى يتمكّن الترجي الرياضي من معرفة منافسه القادم في الدور 16 من نفس المسابقة الإفريقيّة والتي شهدت تغييرات جريئة و متقدّمة في نظامها. إليكم نتائج اليوم من مباريات الدور التمهيدي الاول لدوري ابطال افريقيا : - نابس سبورت (مدغشقر) 2 : 1 تاونشيب رولرز (بوتسوانا) - كوتي دي أور (السيشيل) 0 : 2 كيدوس جيورجيس (إثيوبيا) - توسكر (كينيا) 1 : 1 بورت لويس 2000 (الموريشيس) - زاناكو (زامبيا) 0 : 0 آرمى باتريوريتيك (رواندا ) - رويال ليبوباردس (سوازيلاند) 0 : 1 فيتا كلوب (جمهورية الكونجو) - زيماموتو (زنجبار) 2 : 1 فيروفيارو بيرا (موزمبيق) - ليولي (ليسوثو) 0 : 0 سونترال أفريكان سوسيتي يونايتد (زمبابوي) - سانت لويزيان (جزر الرينيون) 2 : 1 بيدفيست ويتس (جنوب أفريقيا ) - رايل كلوب دو كاديوغو (بوركينا فاسو) 3 : 0 ديابل نوار (الكونجو) - جوهانسينس (سييرا ليون) 1 : 1 الفتح الرباطي (المغرب) - مونانا (الجابون) 2 : 0 فيتالو (بوروندي ) - ريال باماكو (مالي) 0 : 0 ريفرز يونايتد (نيجيريا) - جوري (السنغال) 0 : 0 حوريا (غينيا)  - تاندا (ساحل العاج) 3 : 0 آس فان (النيجر)
